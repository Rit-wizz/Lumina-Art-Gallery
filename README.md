# 🎨 Lumina Art Gallery

## Overview

**Lumina Art Gallery** is a web-based art gallery application designed to provide an immersive digital platform for discovering and appreciating various forms of visual art. The application showcases a curated collection of **paintings**, **photographs**, and **sculptures**, allowing users to explore artworks through an intuitive and visually appealing interface.

The project aims to bridge the gap between artists and art enthusiasts by offering an organized and accessible online gallery experience.

---

## Features

* 🖼️ Browse a diverse collection of paintings.
* 📷 Explore professional photography galleries.
* 🗿 View detailed sculpture collections.
* 👤 User registration and secure login system.
* 🔍 Search and explore artworks by category.
* 📱 Responsive design for desktop, tablet, and mobile devices.
* 🎨 Modern and user-friendly interface.
* 📂 Organized galleries with high-quality images.
* ❤️ User-friendly navigation and seamless browsing experience.

## Gallery Categories

### Paintings

Discover a wide variety of paintings ranging from classical masterpieces to modern artistic creations.

### Photography

Experience stunning landscape, portrait, wildlife, architectural, and street photography captured by talented photographers.

## Installation

1. Clone the repository.

```
git clone https://github.com/your-username/Lumina-Art-Gallery.git
```

2. Open the project in Eclipse.

3. Configure Apache Tomcat.

4. Run the project on the server.

5. Open your browser and visit:

```
http://localhost:8080/Lumina-Art-Gallery
```
